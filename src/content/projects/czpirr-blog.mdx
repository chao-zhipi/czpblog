czpirr Blog 是我当前正在定制的个人技术博客。

它的目标不是做成复杂的门户，而是提供一个稳定、清晰、适合长期积累的写作空间，用来记录 Java 后端、Agent 开发、科研学习和项目实践。

## 项目定位

这个站点会承担三个角色：

- 技术博客：沉淀工程问题、学习笔记和实践复盘；
- 项目归档：展示个人项目，并保留项目背景与实现细节；
- 科研记录：整理论文阅读、实验设计和阶段性思考。

## 当前状态

目前站点已经完成基础定制，包括首页文案、个人介绍、项目入口和第一篇博客文章。

后续会继续补充真实项目、技术文章、科研笔记，以及更适合自己使用的分类和标签结构。

## 后续计划

接下来会优先完善这些内容：

- 增加 Java 后端专题文章；
- 记录 Agent 开发实验和工具链；
- 整理论文阅读与实验复盘；
- 为已完成或正在开发的项目补充独立项目页。
